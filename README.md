# Levi Premer Personal Website

Static personal academic and professional website for Levi Premer.

## Structure

- `index.html` contains the page content and section order.
- `assets/css/styles.css` contains all visual styling and responsive behavior.
- `assets/media/headshot.jpg` is the optimized headshot used in the About section.
- `assets/docs/levi-premer-cv.docx` is the downloadable CV linked from the site.
- `PROJECT_BRIEF.md` documents the site goals, audience, tone, and future content plan.

The site is GitHub Pages compatible and does not require a build step.
